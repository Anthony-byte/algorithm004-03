### Linked List

#### 各种操作的时间复杂度

|  操作   | 时间复杂度 |
| :-----: | :--------: |
| prepend |    O(1)    |
| append  |    O(1)    |
| lookup  |    O(n)    |
| insert  |    O(1)    |
| delete  |    O(1)    |

#### 链表的优化

**核心思想：升维思想，空间换时间**

解决方法：增加多级索引，log(2n)个索引
